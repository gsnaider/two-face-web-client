Two-Face Inpainting

 Two-Face Inpainting es una aplicación para completar regiones de imágenes faciales.
La aplicación recibe dos imágenes de la misma persona: una imagen a ser rellenada, y otra imagen para usar de referencia.
En base a la información de ambas imágenes, la aplicación completa la región faltante de la primer imagen.
